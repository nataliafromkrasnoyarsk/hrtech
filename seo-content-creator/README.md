# SEO Content Creator Plugin v2.0

Интерактивный агент для создания SEO-статей с пошаговым заполнением параметров.

## Что делает плагин

Проводит пользователя через **7-шаговый wizard**, собирая параметры для генерации качественной SEO-статьи.

## Быстрый старт

### Использование (Claude Code)

Команда уже интегрирована в Claude Code. Просто запустите:

```bash
/seo-create-article
```

### Ручная установка

Если команда не доступна, скопируйте файл в директорию команд:

```bash
cp .claude/commands/seo-create-article.md ~/.claude/commands/
```

Или для проекта:

```bash
mkdir -p .claude/commands
cp seo-content-creator/commands/create-article.md .claude/commands/seo-create-article.md
```

## Пример диалога

```
Агент: Привет! Я помогу создать SEO-статью.
       Шаг 1/7: Введите тему статьи:

Вы:    Как выбрать CRM для малого бизнеса

Агент: Тема сохранена.
       Шаг 2/7: Введите ключевые слова или "авто":

Вы:    авто

Агент: Подобрал ключи:
       - crm для малого бизнеса
       - выбор crm системы
       - crm 2024
       Подходит? (да/нет)

Вы:    да

Агент: Шаг 3/7: Выберите режим:
       1. balanced (стандартный)
       2. deep_expert (технический)
       3. light (упрощённый)
       ...
```

## Навигация

В любой момент можно использовать команды:

| Команда | Действие |
|---------|----------|
| `назад` | Вернуться на шаг назад |
| `пропустить` | Пропустить опциональный шаг |
| `статус` | Показать текущие параметры |
| `начать заново` | Перезапустить wizard |
| `помощь` | Показать справку |

## Параметры

### Режимы (Step 3)

| Режим | Описание |
|-------|----------|
| `balanced` | Стандартный (по умолчанию) |
| `deep_expert` | Технический экспертный |
| `light` | Упрощённый для всех |
| `strict_seo` | Максимум SEO |
| `brand_heavy` | Акцент на бренд |

### Тон (Step 4)

| Тон | Описание |
|-----|----------|
| `expert` | Профессиональный |
| `neutral` | Нейтральный |
| `technological` | Технический |

### Аудитория (Step 4)

| Сегмент | Описание |
|---------|----------|
| `SMB` | Малый/средний бизнес |
| `enterprise` | Крупный бизнес |
| `developers` | Разработчики |
| `marketers` | Маркетологи |
| `general` | Широкая аудитория |

### Ограничения (Step 6)

- `no_promo` — без рекламы
- `no_comparison` — без сравнений
- `no_hype` — без преувеличений
- `no_pricing` — без цен

## Результат

После завершения wizard вы получите:

```
Meta-данные
├── Title: SEO заголовок (до 60 символов)
├── Description: Мета-описание (150-160 символов)
└── URL: seo-friendly-url

Варианты H1
├── 1. Вариант с ключом
├── 2. Вариант с вопросом
└── 3. Вариант с годом

Статья
└── Полный текст в Markdown

Статистика
├── Слов: 2500
├── Ключей: 15
└── LSI: 20

Заметки для редактора
```

- `/seo-create-article` — запуск SEO wizard

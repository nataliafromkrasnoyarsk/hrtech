# VK WorkSpace SEO Agent

Интерактивный визард для создания SEO-оптимизированных статей для блога VK WorkSpace (workspace.vk.ru/blog).

## Запуск

```
/seo-article-vk-workspace
```

Или с темой:

```
/seo-article-vk-workspace Российские аналоги Microsoft Teams
```

## Структура

```
vk-workspace-seo-agent/
├── .claude/
│   ├── agents/
│   │   └── vk-workspace-seo-wizard.md    # Агент-визард (12 шагов)
│   └── skills/
│       └── vk-workspace-knowledge/
│           └── SKILL.md                   # База знаний VK WorkSpace
└── README.md
```

## Workflow (12 шагов)

| # | Шаг | Тип | Описание |
|---|-----|-----|----------|
| 1 | Тема статьи | ОБЯЗАТ. | Тип и тема: обзор, гайд, образовательная, продуктовая |
| 2 | Продукт | ОБЯЗАТ. | Почта, Календарь, Диск, Документы, ВКС, Мессенджер, Проекты, Доска, Суперапп, платформа |
| 3 | Роль эксперта | ОБЯЗАТ. | ИТ-директор / Продуктивность / ИБ / Цифровая трансформация / Универсальный |
| 4 | Аудитория | ОБЯЗАТ. | ИТ-директора / Сисадмины / SMB / Разработчики / Широкая |
| 5 | Доп. информация | ОПЦИОН. | Факты, кейсы, цитаты |
| 6 | Ключевые слова | ОБЯЗАТ. | Авто или вручную |
| 7 | Режим | ОБЯЗАТ. | Balanced / Deep Expert / Light / Strict SEO / Brand |
| 8 | Тон | ОБЯЗАТ. | Expert / Neutral / Technological |
| 9 | Структура | ОПЦИОН. | Авто или своя |
| 10 | Подтверждение | — | Сводка параметров |
| 11 | Генерация | — | Статья 1500-3000 слов |
| 12 | Доработка | ОПЦИОН. | Правки, сохранение MD/HTML |

## База знаний

Скилл `vk-workspace-knowledge` содержит:

- Каталог всех продуктов VK WorkSpace (13 сервисов)
- Целевые аудитории (5 сегментов: от микро до энтерпрайз)
- Конкуренты и позиционирование
- Безопасность и compliance
- Терминология и правила написания
- SEO-стратегия и кластеры ключевых слов
- Tone of Voice и стиль блога
- Типы статей и шаблоны структур
- Режимы генерации

## Выходные файлы

Статьи сохраняются в:
```
content-hub/vk-workspace-articles/drafts/{YYYY-MM-DD}-{slug}.md
content-hub/vk-workspace-articles/published/{YYYY-MM-DD}-{slug}.md
```

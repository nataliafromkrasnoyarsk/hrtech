# База знаний VK WorkSpace для SEO-статей

## 1. О платформе

**VK WorkSpace** — платформа для корпоративных коммуникаций и совместной работы команд любого масштаба, где все сервисы интегрированы между собой и доступны через сквозные сценарии в едином супераппе.

### Ключевые факты
- Российский вендор корпоративного ПО, 25-летний опыт VK в развитии интернет-сервисов
- Топ-3 среди производителей отечественного ПО
- 20+ продуктов портфеля входят в реестр российского ПО
- Работает на рынках РФ, Казахстана, Узбекистана
- Лидер рынка On-Premises решений по версии Json Partners
- №1 в рейтинге унифицированных коммуникаций CNews
- Лучшее решение для цифровой экономики — ComNews Awards

### RTB (Reasons To Believe)
1. **Удобно:** все нужные сервисы в одном приложении (в т.ч. почта), прошиты сквозными сценариями
2. **Надежно:** российский разработчик, доверяют крупнейшие компании страны
3. **Для бизнесов любого размера:** в облаке по подписке (SaaS), по подписке в дата-центрах заказчика (Hosted), срочные лицензии (On-Premise)
4. **Безопасность:** защита корпоративного уровня (один суперапп проще обезопасить, чем распределенные сервисы)

### Варианты развертывания
- **On-Cloud (SaaS)** — облачная версия, быстрый старт, оплата по подписке
- **On-Premises** — развертывание в контуре заказчика, от 2000 пользователей
- **Hosted** — подписка в дата-центрах заказчика
- **ПАК (Программно-аппаратный комплекс)** — готовое комплексное решение, включает серверное и сетевое оборудование, ОС, ПО, поддержку

## 2. Каталог продуктов

### 2.1. Суперапп VK WorkSpace
**Позиционирование:** Корпоративное приложение для безопасного доступа к сервисам VK WorkSpace с любых устройств, в котором все сервисы интегрированы между собой.

**Ключевые возможности:**
- Единый интерфейс и единая авторизация
- Кросс-сценарии: перемещение между почтой, мессенджером, календарем, ВКС без потери контекста
- Модульность: администратор настраивает доступные модули
- Оффлайн-режим: работа с письмами и календарем без сети
- Безопасность: шифрование данных, MAM (Mobile Application Management), запрет скриншотов, контроль рутованных устройств
- Интеграция корпоративных сервисов через мини-приложения
- Совместимость: Windows, macOS, Linux (Astra Linux, РЕД ОС), iOS, Android, Huawei, Web

### 2.2. Почта VK WorkSpace
**Позиционирование:** Корпоративная почта для бизнеса любого масштаба, обеспечивающая надежное управление перепиской, высокую безопасность и удобство администрирования.

**Ключевые возможности:**
- Поиск по нескольким параметрам, делегирование прав, общие ящики
- Группировка писем в треды, вложенные папки, фильтры, пересылка
- Отмена/отзыв писем, отложенная отправка, отчет о прочтении
- Массовая миграция из любых почтовых систем
- Отказоустойчивость на уровне хранилищ, сервисов и СУБД
- Геораспределенная почта, катастрофоустойчивость (active-active, 2 ЦОД + witness)
- Интеграция с DLP (InfoWatch, Solar Dozor, SearchInform, Zecurion)
- Антивирус: Dr.Web (облако), Kaspersky KLMS/KSMG (On-Premise)
- Протоколы: IMAP/SMTP, Kerberos, CardDav, CalDAV, SSO
- Клиенты: Суперапп, MS Outlook, Р7-Органайзер, Web

### 2.3. Календарь VK WorkSpace
**Позиционирование:** Календарь для бизнеса, интегрированный с корпоративной почтой, упрощает организацию встреч и управление рабочим временем.

**Ключевые возможности:**
- Создание событий из писем и мессенджера
- Просмотр занятости участников, бронирование переговорных
- Двусторонняя синхронизация с Exchange, CalDAV
- Режимы: Расписание, День, Неделя, Месяц
- Делегирование доступа секретарю, печать расписания
- Оффлайн-режим, уведомления, прикрепление файлов

### 2.4. Диск VK WorkSpace
**Позиционирование:** Надежное файловое хранилище для совместной работы команд, обеспечивающее гибкое управление доступом и защиту данных.

**Ключевые возможности:**
- Централизованное хранение и управление файлами
- Общие папки, настройка прав доступа (пользователи, группы)
- Внешние ссылки с паролем, ограничением по времени/скачиваниям
- Миграция из Microsoft OneDrive, Google Disk, NextCloud
- Интеграция с DLP-системами, SSO через Kerberos
- Хранение в распределенных ЦОД на территории РФ или в контуре заказчика

### 2.5. Документы VK WorkSpace
**Позиционирование:** Сервис для создания и совместного редактирования текстов, таблиц и презентаций.

**Ключевые возможности:**
- Форматы: docx, pdf, xlsx, pptx
- Совместная работа в реальном времени
- История версий, персональный и общий доступ по ссылке
- Интеграция с Диском VK WorkSpace
- Технологическое партнерство с Р7-Офис
- WOPI-совместимость (Р7-Офис, МойОфис)
- Входит в реестр российского ПО

### 2.6. Мессенджер VK WorkSpace
**Позиционирование:** Корпоративный мессенджер для мгновенного общения в чатах и каналах, с привычным интерфейсом, повышающий эффективность командной работы и безопасность коммуникаций.

**Ключевые возможности:**
- Публичные/приватные чаты и каналы
- Треды (ветки обсуждений), реакции, закрепленные сообщения
- Запланированные сообщения, частичное цитирование
- Папки чатов, архив, избранное
- Чат-боты через Bot API (Go, Python, JavaScript)
- Интеграция с DLP: InfoWatch, Solar Dozor, SearchInform, Zecurion
- Федерация между On-Premise инсталляциями
- VIP-уровни: прямые сообщения руководству от определенных ролей

### 2.7. Видеоконференции VK WorkSpace
**Позиционирование:** Аудио- и видеозвонки с участием до 500 пользователей, обеспечивающие удобство и безопасность коммуникаций.

**Ключевые возможности:**
- Звонки до 500 участников
- Режим вебинара (300+ человек) с ролями: организатор, модератор, слушатель
- Демонстрация экрана (до 4K), запись, чат в звонке
- FullHD-стриминг, адаптивное эхо- и шумоподавление
- Интеграция с SIP АТС и SIP видеозвонками
- ПО для управления переговорными комнатами
- Гостевой доступ по ссылке для партнеров
- Зал ожидания, управление устройствами участников

### 2.8. Доска VK WorkSpace
**Позиционирование:** Интерактивный сервис для управления проектами и совместной работы, от заметок до канбан-досок и интеллект-карт. Замена Miro.

**Ключевые возможности:**
- Инструменты: фигуры, коннекторы, стикеры, маркеры, таблицы, текст, стрелки, фреймы
- Шаблоны: Командное ретро, Mind Map, SWOT-анализ, Kanban, Agile roadmap, Диаграмма Ганта
- Создание собственных шаблонов
- Комментарии, реакции, голосование, таймер
- Режим презентации
- Массовая миграция из Miro
- Права: читатель, редактор, комментатор, администратор
- Уровни доступа: для всех, в пределах домена, только для владельца
- SSO, аудит-логи, мобильная версия
- Покрывает 80% востребованной функциональности Miro
- Нет ограничений на количество досок и команд

### 2.9. Проекты VK WorkSpace
**Позиционирование:** Корпоративное решение для управления проектами и задачами в экосистеме VK WorkSpace: контроль сроков и загрузки, сквозная аналитика и автоматизация процессов.

**Ключевые возможности:**
- Гибкие представления: канбан, таблица, таймлайн/Гант
- Цели/OKR и приоритизация
- Встроенные "Страницы" для документов с комментариями
- Спринты, бэклог, Scrum-инструменты
- Автоматизация процессов
- Интеграции с Git/CI/CD и корпоративными системами
- SSO/AD, роли и аудит изменений
- Миграция из Jira и других систем
- Учет загрузки команд
- On-Premise развертывание

### 2.10. Задачи VK WorkSpace
**Позиционирование:** Инструмент для планирования рабочих и личных задач.

**Возможности:** постановка задач из чата, приоритет, ответственный, дедлайн, отслеживание выполнения.

### 2.11. Опросы VK WorkSpace
**Позиционирование:** Сервис для сбора и анализа обратной связи сотрудников.

**Возможности:** анонимные/неанонимные опросы, таргетирование аудитории, типы вопросов (выбор, шкала, открытые), выгрузка отчетов, анализ в реальном времени.

### 2.12. Оргструктура VK WorkSpace
**Позиционирование:** Сервис для быстрого поиска коллег по подразделению или отделу.

**Возможности:** структура организации и группы компаний, профили сотрудников и подразделений, интеграция с HCM-системами.

### 2.13. Заметки VK WorkSpace
**Позиционирование:** Сервис для создания заметок, планирования и совместной работы.

**Возможности:** хранение информации, чек-листы, таблицы, совместная работа с коллегами.

## 3. Целевые аудитории

### 3.1. Микро- и малый бизнес (1-50 сотрудников)
- **ЛПР:** Руководитель компании, директор
- **ЛВПР:** Специалисты ИТ, системные администраторы
- **Факторы выбора:** простота, скорость внедрения, минимум обучения
- **Модель:** преимущественно On-Cloud
- **Ключевые аргументы:** единая платформа для всех задач, экономия ресурсов (не нужно серверное оборудование), гибкая тарификация, доступность с любых устройств, интуитивное администрирование

### 3.2. Малый и средний бизнес (51-200 сотрудников)
- **ЛПР:** Руководитель компании, директор
- **ЛВПР:** Специалисты ИТ, системные администраторы
- **Факторы выбора:** проверенные продукты, безопасность данных, серверы в РФ, стоимость
- **Модель:** преимущественно On-Cloud
- **Ключевые аргументы:** продукт от технологического лидера, масштабируемость, тестовый доступ 30 дней, быстрый старт

### 3.3. Средний и крупный бизнес (201-1000 сотрудников)
- **ЛПР:** Руководитель компании, руководитель ИТ-подразделения
- **ЛВПР:** Специалисты ИТ, системные администраторы
- **Факторы выбора:** российское происхождение приоритет, безопасность важнее стоимости
- **Модель:** On-Cloud (70%) + On-Premises (30%)
- **Ключевые аргументы:** реестр российского ПО, ФЗ-152, масштабируемость, единое пространство

### 3.4. Крупный бизнес (501-3000 сотрудников)
- **ЛПР:** CTO, CIO, руководитель ИТ-департамента
- **ЛВПР:** Руководитель по ИБ, операционный/генеральный директор, директор по закупкам, CFO
- **Факторы выбора:** максимальная безопасность, многофункциональность, техподдержка
- **Модель:** On-Cloud (65-70%) + On-Premises (30-35%)
- **Ключевые аргументы:** ФЗ-152, отказоустойчивость при высоких нагрузках, UX/UI, 25-летний опыт, интеграция с DLP/SIEM, миграция из других систем

### 3.5. Энтерпрайз (3001-8000+ сотрудников)
- **ЛПР:** CTO, CIO, руководитель ИТ-департамента
- **ЛВПР:** Руководитель по ИБ, операционный/генеральный директор, директор по закупкам, CFO
- **Факторы выбора:** безопасность, стандарты, масштабируемость, интеграции
- **Модель:** преимущественно On-Premises
- **Ключевые аргументы:** размещение в контуре компании, SSO/AD, аудит, ПАК, катастрофоустойчивость, геораспределенная архитектура

### Отрасли клиентов
Нефтегазовая, промышленность, госсектор, горнодобывающая, транспорт, электроэнергетика, сельское хозяйство, банки и страхование, лизинг, МФО, розничная торговля, дистрибуция, FMCG, e-commerce, телекоммуникации, образование, логистика.

## 9. Tone of Voice и стиль блога

### Характеристики тона
- **Профессиональный, но доступный** — деловой русский без канцеляризмов
- **Консультативный** — позиция эксперта, который помогает разобраться
- **Информативный** — конкретика, цифры, факты вместо общих фраз
- **Нейтральный** — без агрессивной рекламы, VK WorkSpace упоминается органично как решение

### Структурные паттерны статей блога

**Типичная структура SEO-статьи:**
1. **Заголовок H1** — содержит основной ключ, до 70 символов
2. **Лид** — 2-3 предложения с проблемой/статистикой, hook для читателя
3. **Содержание** — якорные ссылки на разделы
4. **Основные разделы (H2/H3):**
   - Определение понятия / постановка проблемы
   - Причины / критерии / этапы
   - Обзор решений / пошаговая инструкция
   - Сравнительные таблицы
   - Органичное упоминание VK WorkSpace как решения
   - Рекомендации по сегментам
5. **FAQ** — 5-8 частых вопросов (для featured snippets)
6. **Заключение** — краткие выводы
7. **CTA** — "Запросите демо" или "Попробуйте бесплатно 30 дней"

### Правила написания
- Абзацы до 3-4 предложений
- Списки и таблицы для сканируемости
- Конкретные примеры и сценарии использования
- Ссылки на авторитетные источники (исследования, регуляторы)
- Не начинать статью со слов "В современном мире" или "В эпоху цифровизации"
- Не использовать штампы: "уникальный", "инновационный", "передовой"
- Интеграция VK WorkSpace — через конкретные сценарии, а не через рекламные блоки
- Meta description: 150-160 символов, содержит ключ и призыв к действию

### Примеры удачных формулировок из блога
- "Jira долгое время была де-факто базовым инструментом"
- "На практике компании не просто переходят с одного сервиса на другой"
- "Представьте: вы наняли отличного кандидата, потратили время и силы — а уже через месяц он уходит"
- "Внедрение цифровых сервисов — уже не просто возможность, а необходимость для бизнеса"

### Типы CTA
- "Узнать подробнее" → ссылка на страницу продукта
- "Задайте вопрос экспертам или запросите демо" → форма обратной связи
- "Попробуйте бесплатно 30 дней" → тестовый доступ
- "Подписаться на блог" → рассылка

- Обзор нескольких решений по категории

- Скриншоты и примеры
- Чек-листы, FAQ

## 11. Режимы генерации

### Balanced (по умолчанию)
Оптимальный баланс SEO и читаемости. Естественное вхождение ключей, фокус на пользу для читателя.

### Deep Expert
Глубокая экспертная статья с техническими деталями, архитектурными схемами, сравнениями API. Для аудитории: ИТ-директора, системные администраторы.

### Light
Легкая, доступная статья для широкой аудитории. Минимум технической терминологии, акцент на бизнес-ценность. Для: руководители, HR, маркетологи.

### Strict SEO
Максимальная SEO-оптимизация: точные вхождения ключей, LSI-слова, оптимальная плотность, структура для featured snippets. Для: максимальный органический трафик.

### Brand
Имиджевая статья с фокусом на позиционирование VK WorkSpace. Сторителлинг, кейсы, цитаты. Для: повышение узнаваемости бренда.

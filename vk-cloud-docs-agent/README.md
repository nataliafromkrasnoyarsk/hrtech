# VK Cloud Docs Agent

Агент для написания технических статей о сервисах VK Cloud на основе официальной документации.

## Описание

**VK Cloud Docs Writer** — специализированный агент, который создаёт качественный технический контент о сервисах VK Cloud. Агент автоматически обращается к официальной документации в репозитории [vk-cs/docs-public](https://github.com/vk-cs/docs-public) для получения актуальной информации.

## Использование

### Через команду

```bash
/write-vk-cloud-article
```

### Через активационные слова

Агент автоматически активируется при упоминании:
- "напиши статью о VK Cloud"
- "статья про облако VK"
- "VK Cloud документация"
- "гайд по VK Cloud"
- "обзор VK Cloud"
- "инструкция VK Cloud"

## Возможности

### Типы статей

| Тип | Описание |
|-----|----------|
| Обзорная статья | Обзор возможностей сервиса для новых пользователей |
| Туториал | Пошаговая инструкция по решению конкретной задачи |
| Сравнение | Сравнение с AWS/GCP/Azure или между сервисами VK Cloud |
| Best Practices | Рекомендации по оптимальному использованию |
| Миграция | Гайд по переносу с других платформ |
| Интеграция | Настройка совместной работы сервисов |

### Поддерживаемые сервисы VK Cloud

**Compute:**
- Cloud Servers (виртуальные машины)
- Kubernetes aaS
- Serverless Containers

**Databases:**
- PostgreSQL, MySQL, MongoDB
- Redis, ClickHouse, Kafka
- И другие managed databases

**Storage:**
- Object Storage (S3-compatible)
- File Storage (NFS/CIFS)
- Backup

**Networks:**
- Virtual Networks, VPC
- Load Balancers
- CDN, DNS, VPN

**Data & ML:**
- Big Data (Hadoop, Spark)
- ML Platform
- Cloud Vision, Cloud Voice

**Security & Monitoring:**
- WAF, DDoS Protection
- Cloud Monitoring
- Cloud Logging

## Примечания

1. **Актуальность:** Документация VK Cloud регулярно обновляется. Агент всегда проверяет текущее состояние в репозитории.

2. **Терминология:** Используются официальные названия сервисов VK Cloud.

3. **Ценообразование:** Агент не указывает конкретные цены — они меняются. Ссылается на официальный калькулятор.

4. **Региональность:** VK Cloud работает в российских регионах. Это учитывается при описании compliance, latency и т.д.

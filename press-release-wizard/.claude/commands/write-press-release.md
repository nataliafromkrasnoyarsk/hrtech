# /press-release-wizard:write-press-release

Запускает интерактивного агента для создания пресс-релиза.

## Usage

```
/press-release-wizard:write-press-release
```

Или просто напишите:
- "Напиши пресс-релиз"
- "Нужен пресс-релиз о [продукте]"
- "Создай релиз"

## Workflow

```
┌─────────────────────────────────────────────────────────┐
│  1. ПРИЁМ МАТЕРИАЛОВ                                   │
│     Загрузите файл, вставьте текст или опишите          │
├─────────────────────────────────────────────────────────┤
│  2. АНАЛИЗ                                             │
│     Агент выделяет ключевые факты и выявляет пробелы    │
├─────────────────────────────────────────────────────────┤
│  3. УТОЧНЕНИЕ                                          │
│     Вопросы по одному для заполнения пробелов           │
├─────────────────────────────────────────────────────────┤
│  4. ГЕНЕРАЦИЯ                                          │
│     Создание текста в стиле VK Cloud                    │
├─────────────────────────────────────────────────────────┤
│  5. ДОРАБОТКА                                          │
│     Корректировка по вашим замечаниям                   │
└─────────────────────────────────────────────────────────┘
```

## Input Options

### 1. Загрузка файла
Поддерживаются: PDF, DOCX, PPTX, TXT, MD

### 2. Текст в чате
Вставьте описание продукта, техническую документацию или черновик.

### 3. Словесное описание
```
Нужен релиз о новом сервисе Cloud Audit. Это сервис для проверки 
безопасности кода, находит уязвимости до продакшена.
```

## Questions Asked

Агент задаёт вопросы только если информация отсутствует в материалах:

| Вопрос | Пример ответа |
|--------|---------------|
| Главная ценность | "Сокращает время разработки в 2 раза" |
| Целевая аудитория | "DevOps-инженеры, разработчики" |
| Цифры и метрики | "10 000 сканирований в день" |
| Спикер для цитаты | "Дмитрий Лазаренко, директор по продукту" |
| Партнёр (если есть) | "QazCloud" или "нет" |
| Сертификаты | "152-ФЗ, ФСТЭК" или "не применимо" |

## Output

### Готовый текст

```markdown
27 июня 2025 • Пресс-релизы

# VK Cloud запустила сервис для повышения безопасности приложений

VK Cloud представила новый сервис — Cloud Audit...

*«Цитата спикера»*, — комментирует **Имя Фамилия, должность**.

**VK Cloud** — платформа с широким набором облачных сервисов...
```

### Статистика
- Слов: ~500
- Цитат: 1-2
- Ссылок: 0-1

## Refinement Commands

После генерации можно дорабатывать:

| Команда | Действие |
|---------|----------|
| `короче` | Сократить на 20% |
| `подробнее` | Добавить деталей |
| `другой заголовок` | 3 варианта |
| `переписать цитату` | Новая версия |
| `добавить цифры` | Усилить фактами |
| `формальнее` / `проще` | Изменить тон |

## Style

Агент пишет в стиле VK Cloud:
- Деловой, но читаемый тон
- Конкретные цифры вместо "значительно"
- Без превосходных степеней
- Короткие абзацы (3-5 предложений)

## Examples

### Минимальный ввод

```
Пользователь: Напиши релиз о запуске VK Data Platform

Агент: Расскажите подробнее о VK Data Platform...
       [серия уточняющих вопросов]
       [генерация текста]
```

### С материалами

```
Пользователь: [загружает презентацию продукта]

Агент: Изучил материалы. Вижу, что это платформа для работы с данными.
       Уточню несколько моментов...
       [1-2 вопроса]
       [генерация текста]
```

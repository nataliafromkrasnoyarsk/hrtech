# VK Tech повысила безопасность и прозрачность разработки в новой версии VK Dev Platform

VK Tech выпустила обновление платформы для разработки программного обеспечения — VK Dev Platform 1.7. В новой версии усилена защита конфиденциальных данных, добавлена единая система мониторинга и обновлено хранилище компонентов для сборки приложений.

VK Dev Platform — единая среда для разработки ПО с готовой инфраструктурой и инструментами для полного цикла создания программных продуктов. Платформа обеспечивает DevSecOps-подход — практику встраивания проверок безопасности в процесс разработки — по умолчанию и поддерживает развёртывание в контуре VK Cloud и VK Private Cloud.

Ключевое изменение — шифрование конфиденциальных данных. Все пароли, ключи доступа и секреты теперь хранятся в зашифрованном виде и недоступны даже при утечке базы данных.

Хранилище артефактов VK Dev Platform, разработанное на базе Nexus — системы управления компонентами для сборки приложений, обновлено до актуальной версии, что повышает отказоустойчивость сервиса. Также автоматизировано управление доступом — права пользователей синхронизируются между платформой и системой тестирования TestIT без участия администратора. Упрощён выход из системы: завершить сессию во всех сервисах можно одним действием.

Для повышения наблюдаемости платформа также получила встроенные инструменты мониторинга: логи и метрики производительности всех сервисов собираются в одном месте, что ускоряет поиск и устранение сбоев.

*«По прогнозам Центра стратегических разработок, рынок безопасной разработки в России вырастет до 60 млрд рублей к 2027 году. Требования к защите данных становятся строже, и мы видим запрос на платформы, где безопасность встроена по умолчанию. В VK Dev Platform 1.7 защита данных и система мониторинга интегрированы в ядро продукта — разработчики могут сосредоточиться на создании продуктов, а не на настройке инфраструктуры»*, — отметил **руководитель направления облачных и дата-сервисов VK Tech Дмитрий Лазаренко**.

**VK Tech** — российский разработчик корпоративного ПО для решения ежедневных задач бизнеса. Портфель VK Tech — это готовая экосистема программных продуктов, которая включает облачную платформу, дата-сервисы, сервисы продуктивности, бизнес-приложения. Продукты VK Tech включены в реестр российского ПО и соответствуют требованиям ФСТЭК, поставляются в формате лицензий (On-Premise) и по подписке (On-Cloud), что делает их доступными для компаний любого масштаба.

**VK Cloud** — платформа с широким набором облачных сервисов и ПО для эффективной разработки и работы с данными для компаний любого масштаба. VK Cloud входит в портфель решений VK Tech и базируется на многолетнем опыте развития интернет-сервисов и технологий на базе открытого кода. Среди продуктов VK Cloud — инфраструктурные и платформенные облачные сервисы, а также комплексные решения, такие как Dev Platform, доступные в Public Cloud, в рамках инсталляции Private Cloud и как софт. Клиенты компании: Битрикс, Газпромнефть, Росатом и другие крупнейшие игроки своих отраслей.

|---------|------------|

## Ключевые особенности этого релиза

### 1. Без буллитов

Все изменения описаны сплошным текстом, а не списком:
- Шифрование → отдельный абзац
- Хранилище артефактов → отдельный абзац
- Мониторинг → отдельный абзац

### 2. Расшифровка терминов

| Термин | Как расшифрован |
|--------|-----------------|
| DevSecOps | "практику встраивания проверок безопасности в процесс разработки" |
| Nexus | "системы управления компонентами для сборки приложений" |
| Секреты | "пароли, ключи доступа и секреты" |
| Наблюдаемость | "логи и метрики производительности" |

### 3. Описание продукта после лида

Второй абзац целиком посвящён описанию VK Dev Platform — что это и как работает.

### 4. Конкретные цифры

- 60 млрд рублей — рынок к 2027 году
- Версия 1.7
- Источник: Центр стратегических разработок

### 5. Два бойлерплейта

Для обновления продукта VK Tech указаны оба бойлерплейта:
- VK Tech (основная компания-разработчик)
- VK Cloud (платформа, где работает продукт)

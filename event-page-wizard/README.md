# Event Page Wizard Plugin

Интерактивный агент для создания страниц мероприятий в стиле VK Cloud.

## 🎯 Что делает плагин

Проводит пользователя через **10-шаговый wizard**, собирая данные для генерации страницы мероприятия (вебинара, конференции, митапа).

## 🚀 Быстрый старт

### Установка

```bash
/plugin marketplace add levashove/agents
/plugin install event-page-wizard
```

### Запуск

```bash
/event-page-wizard:create-event
```

Или просто напишите:
```
Создай страницу вебинара
```

## 📂 Типы мероприятий

| Тип | Описание | Формат |
|-----|----------|--------|
| вебинар | Онлайн-трансляция | online |
| конференция | Масштабное мероприятие | hybrid |
| митап | Неформальная встреча | offline |
| воркшоп | Практическое занятие | offline |
| партнёрское | Совместное мероприятие | varies |

## 🧭 Навигация

| Команда | Действие |
|---------|----------|
| `назад` | Вернуться на шаг |
| `пропустить` | Пропустить опциональный шаг |
| `статус` | Показать текущие данные |
| `начать заново` | Перезапустить wizard |

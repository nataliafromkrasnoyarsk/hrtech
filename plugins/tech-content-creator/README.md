# Tech Content Creator Plugin

**Elite content creation system for high-tech companies** — создание статей мирового уровня для ведущих бизнес и технологических медиа.

## Обзор / Overview

Comprehensive content creation workflow от исследования до публикации, включающий:

1. **Research & Trend Analysis** — Исследование трендов и анализ возможностей
2. **Content Ideation** — Генерация идей с IMPACT-оценкой
3. **Professional Drafting** — Профессиональное написание статей
4. **Rigorous Fact-Checking** — Тщательная проверка фактов
5. **Expert Editing** — Экспертное редактирование
6. **SEO Optimization** — Оптимизация для поисковых систем и распространения

## Комплексные навыки / Comprehensive Skills

### 1. tech-trends-research
Методология исследования технологических трендов, анализ рынка, идентификация контент-возможностей.

**Содержание**:
- Industry intelligence sources (Gartner, Forrester, IDC)
- Search trend analysis techniques
- Competitive content analysis frameworks
- Audience intelligence gathering
- IMPACT scoring methodology
- Trend evaluation and prioritization

**Шаблоны**: Trend research brief, competitive analysis, keyword research report

### 2. enterprise-storytelling
Корпоративное повествование для высокотехнологичных компаний — трансформация сложных технических концепций в compelling narratives.

**Содержание**:
- Narrative arc for technical content
- Human element in B2B storytelling
- Stakes and tension creation
- Four storytelling frameworks (CSR, Contrarian, Journey, How-To)
- Storytelling techniques and writing for different audiences
- Cultural considerations (Russian vs. English)

**Шаблоны**: Case study template, opinion piece template, how-to guide template

### 3. technical-writing-standards
Стандарты технического письма для ясного, точного и профессионального контента на русском и английском.

**Содержание**:
- The Three Cs: Clear, Concise, Correct
- Sentence construction and paragraph structure
- Technical terminology guidelines
- Code examples and API documentation
- Russian language standards
- Formatting and presentation

**Справочники**: Style guide checklist, terminology glossary (RU-EN)

### 4. fact-checking-methodology
Rigorous fact-checking methodology для обеспечения точности и credibility технического контента.

**Содержание**:
- Three-layer verification model
- Source hierarchy (Tier 1-4)
- Verification checklists (statistics, technical claims, code, quotes)
- Common fact-checking scenarios
- International and Russian market verification
- Tools and techniques

**Шаблоны**: Fact-check report template, source evaluation guide

### 5. editorial-excellence
Advanced editorial techniques для трансформации good content в exceptional published articles.

**Содержание**:
- The editorial process (structural, paragraph, sentence, word-level)
- Advanced editing techniques
- Common content problems and fixes
- Editorial standards by publication type
- Russian vs. English editorial considerations
- Giving editorial feedback

**Справочники**: Editing checklist, before-after examples

### 6. media-publishing-guidelines
Comprehensive guidelines для публикации в ведущих бизнес и технологических медиа.

**Содержание**:
- Publication landscape overview (Tier 1-4 outlets)
- Publication-specific requirements (Forbes, TechCrunch, HBR, Habr, VC.ru, etc.)
- Pitching to editors (structure, elements, common mistakes)
- Editorial relationships building
- Legal and ethical considerations
- Content adaptation strategies

**Справочники**: Publication directory, pitch templates

## Команды / Commands

### /article-workflow
**Complete article creation workflow** от research до publication-ready content.

**Процесс**:
1. Research & Strategy (tech-content-strategist)
2. Content Ideation (tech-content-strategist)
3. Article Drafting (tech-content-writer)
4. Fact-Checking (tech-fact-checker)
5. Editing (content-editor)
6. SEO & Distribution Optimization (seo-content-optimizer)

**Опции**: Fast track (1 week), Standard (2 weeks), Deep-dive (4 weeks)

**Output**: Complete content package with article, metadata, images, social assets, and process reports.

### /trend-research
**Dedicated trend research command** для comprehensive analysis и content opportunity identification.

**Возможности**:
- Market intelligence gathering
- Competitive content analysis
- Audience intelligence and pain points
- IMPACT framework scoring
- Content opportunity matrix

**Output**: Detailed trend research report, competitive analysis, keyword research.

### /draft-article
**Focused drafting command** для creating article drafts from content brief or topic.

**Процесс**:
1. Input gathering (brief or interactive)
2. Outline creation and approval
3. Draft creation
4. Self-review
5. Delivery with editorial notes

**Output**: Complete draft, outline, editorial notes, metadata.

## Быстрый старт / Quick Start

### Создание статьи с нуля

```bash
# Запустить полный workflow
/article-workflow

# Система проведет через все этапы:
# 1. Research и выбор темы
# 2. Генерация идей
# 3. Написание черновика
# 4. Проверка фактов
# 5. Редактирование
# 6. SEO-оптимизация
```

### Только исследование трендов

```bash
# Исследовать тренды для quarterly planning
/trend-research "AI and Machine Learning Infrastructure"

# Output: Comprehensive trend report с IMPACT scores и content ideas
```

### Создать черновик по существующему brief

```bash
# Использовать существующий content brief
/draft-article --brief=path/to/brief.md

# Или интерактивно
/draft-article
```

## Use Cases / Примеры использования

### Use Case 1: Quarterly Content Planning
**Цель**: Plan Q1 content strategy around emerging cloud-native trends
**Процесс**:
1. Run `/trend-research "Cloud Native Technologies 2024"`
2. Review IMPACT scores and select top 3 trends
3. Create content calendar with publication targets
4. Execute `/article-workflow` for priority articles

**Результат**: Strategic content pipeline aligned with market trends

### Use Case 2: Thought Leadership for Forbes
**Цель**: Publish executive-focused AI governance opinion piece in Forbes
**Процесс**:
1. Run `/article-workflow` with Forbes as target
2. Focus on business impact, not technical details
3. Include data from research reports
4. Editorial review against Forbes standards
5. Pitch with compelling business angle

**Результат**: Forbes contributor article establishing thought leadership

### Use Case 3: Technical Tutorial for Habr
**Цель**: Deep-dive Kubernetes networking tutorial for Russian developer community
**Процесс**:
1. Run `/draft-article --format=tutorial --publication=habr --language=russian`
2. Include detailed code examples and configurations
3. Technical fact-checking of all code
4. Russian language editorial review
5. Publish on Habr with community engagement

**Результат**: High-quality technical tutorial with strong community engagement

### Use Case 4: Customer Case Study
**Цель**: Enterprise cloud migration case study for TechCrunch and company blog
**Процесс**:
1. Gather customer data, metrics, quotes
2. Run `/draft-article --format=case-study`
3. Focus on Challenge-Solution-Results structure
4. Rigorous fact-checking of all metrics
5. Create two versions: TechCrunch (news angle) and blog (detailed technical)

**Результат**: Multi-channel case study with customer validation

## Best Practices / Лучшие практики

**Перед началом**:
- Определите business objectives для контента
- Идентифицируйте target audience и publications
- Соберите supporting materials (data, case studies, quotes)
- Установите realistic timeline

**Во время работы**:
- Следуйте процессу — каждый этап добавляет ценность
- Не пропускайте fact-checking — credibility критична
- Вовлекайте SMEs на ранних стадиях
- Итерируйте на основе feedback

**После публикации**:
- Отслеживайте performance metrics
- Вовлекайтесь в comments и discussions
- Документируйте learnings для улучшения процесса
- Поддерживайте отношения с editors

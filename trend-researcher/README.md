# Trend Researcher

Исследование технологических трендов и стратегическое планирование контента для ведущих бизнес и технологических медиа.

## Что делает

- Анализ технологических трендов и рыночных возможностей
- IMPACT-скоринг тем для приоритизации контента
- Конкурентный анализ контента
- Подготовка питчей для изданий
- Аудитория и pain points mapping

## Быстрый старт

```bash
/trend-research "Cloud Native Technologies 2024"
```

## Агент

### Tech Content Strategist

Элитный стратег контента для высокотехнологичных компаний. Специализируется на создании thought leadership контента для топовых изданий (Forbes, TechCrunch, Wired, HBR, Ведомости, РБК, Хабр, VC.ru).

**Возможности:**
- Trend research and competitive analysis
- IMPACT scoring framework для приоритизации тем
- Content ideation and strategic planning
- Publication matching and pitch development
- Audience intelligence and pain point mapping

## Skill: tech-trends-research

Методология исследования технологических трендов:

- Industry intelligence sources (Gartner, Forrester, IDC)
- Search trend analysis techniques
- Competitive content analysis frameworks
- IMPACT scoring methodology
- Trend evaluation and prioritization

## IMPACT Framework

| Критерий | Описание |
|----------|----------|
| **I**ndustry Relevance | Релевантность для индустрии |
| **M**arket Timing | Своевременность |
| **P**ublication Fit | Соответствие изданию |
| **A**udience Interest | Интерес аудитории |
| **C**ompetitive Gap | Конкурентный разрыв |
| **T**hought Leadership | Потенциал для лидерства мнений |
